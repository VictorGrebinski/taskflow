# TaskFlow

Aplicação web Full Stack para gerenciamento de tarefas desenvolvida com React.js, TypeScript, NestJS e PostgreSQL.

## 🚀 Deploy

Frontend: https://taskflow-jet-five.vercel.app

---

## 📌 Funcionalidades

- Criar tarefas
- Editar tarefas
- Remover tarefas
- Marcar tarefas como concluídas
- Interface responsiva
- Feedback visual para ações do usuário
- Loading states durante carregamentos
- Integração com API REST

---
